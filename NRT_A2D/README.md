Purpose: NRT A2D stands for Near Real time Analouge to Digital. The purpose of this program is to automatically save CSV's from the oscilloscope's webpage.
Setup: In order for this program to work, the computer must be connected to the oscilloscope with an ethernet cable. Additionally, pywinauto must be downloaded on your computer.
Finally, you must visit http://(ip address of oscilloscope) and be on the save page.
